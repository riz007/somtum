# somtum

## 1.0.0

### Major Changes

- 5ad7bc4: Robust multi-device sync, automated re-indexing, and enhanced system diagnostics.

  This release marks the transition of Somtum from an experimental tool to a production-ready memory layer for Claude Code.
  - **Robust Sync:** Implemented hostname-aware syncing to prevent data loss across multiple devices.
  - **Auto-indexing:** New memories are now automatically embedded for semantic search immediately after capture.
  - **Doctor Fixes:** Corrected migration checks and added comprehensive system health diagnostics.
  - **Branding:** Added a new Thai-style colored ASCII logo and polished CLI interface.
  - **Testing:** Significant increase in CLI test coverage to ensure stability.
